JOIN == , == CROSS JOIN == FULL JOIN
CROSS JOIN返回两表的数据行的乘积的记录
一下四行返回结果一样
SELECT * FROM TABLE_A CROSS JOIN TABLE_B
SELECT * FROM TABLE_A FULL JOIN TABLE_B
SELECT * FROM TABLE_A JOIN TABLE_B
SELECT * FROM TABLE_A, TABLE_B

OUT JOIN 返回一张表的所有行，并带有来自另外一张表的条件相符的行
LEFT OUTER JOIN == LEFT JOIN
RIGHT OUTER JOIN == RIGHT JOIN

INNER JOIN 两表都存在的数据

UNION合并几个结果集，并且排除重复数据

GROUP BY一般和聚合函数SUM/AVG一起使用
SELECT CID, AVG(SCORE) FROM SCORE AS S GROUP BY S.CID;
SELECT 
	C.CNAME, AVG(S.SCORE) 
FROM 
	SCORE AS S 
INNER JOIN 
	COURSE AS C 
ON 
	C.CID=S.CID 
GROUP BY 
	C.CID;
