grep
grep -in 'MAD' file.txt --color=auto
显示file.txt中有MAD的行，忽略大小写

-i忽略大小写
-n显示行号
-v反转
--color=auto显示颜色


cat
cat -An ~/env/tiny
-A -vET的组合
-n 显示所有行的行号，包括空白行
-v 显示特殊字符
-E 显示换行符为$
-T 显示Tab为^I
-b 显示行号，不显示空白行的行号


tac
将cat取反


less
less ~/env/tiny
空白键 向下翻动一页；
pagedown 向下翻动一页；
pageup 向上翻动一页；
/字串 向下搜寻『字串』的功能；
?字串 向上搜寻『字串』的功能；
n 重复前一个搜寻 (与 / 或 ? 有关！)
N 反向的重复前一个搜寻 (与 / 或 ? 有关！)
q 离开 less 这个程序；



head
显示前几行
head -n 15 ~/env/tiny
-n 后面空格接数字，代表显示几行

默认显示前面十行
如果后面100行的数据都不列印，用负数
head -n -100 ~/env/tiny



tail
显示从尾巴开始数多少行数据
-n 用法和head一样
tail -n +10 ~/env/tiny
文件一开头10行不显示

-f 监控改文件
tail -f ~/env/tiny
你想要让该文件有数据写入时就立刻显示到屏幕上，就利用 -f 这个选项，他可以一直侦测/var/log/messages这个文件，新加入的数据都会被显示到屏幕上。 直到你按下[crtl]-c才会离开tail


nohup command &      后台运行某程序
nohup command        前景运行某程序


2>&1 标准输出1和标准错误输出2

查找文件
whereis -bmsu file_name
-b    :只找 binary 格式的檔案
-m    :只找在說明檔 manual 路徑下的檔案
-s    :只找 source 來源檔案
-u    :搜尋不在上述三個項目當中的其他特殊檔案

或者
locate -i keyword
-i  ：忽略大小寫的差異；
-r  ：後面可接正規表示法的顯示方式

或者
find -name file_name

或者
grep -r -i file_name ./
-r 递归
-i 忽略大小写

打包时排除部分文件
tar -cvf newTar.tar /folder-name --exclude file-name/sub-folder-name
